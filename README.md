黑马旅游网  
===

## 一.技术选型  
 
 #### 1. Web层  
 
    1. Servlet : 前端控制器  
    2. html : 视图  
    3. Filter : 过滤器  
    4. BeanUtils : 数据封装  
    5. Jackson : json序列化  
 #### 2. Service层  
 
    1. Javamail : java发送邮件工具  
    2. Redis : nosql内存数据库  
    3. Jedis : java的redis客户端  
		
 #### 3. Dao层  
 
    1. Mysql : 数据库  
    2. Druid : 数据库连接池  
    3. JdbcTemplate : jdbc的工具
